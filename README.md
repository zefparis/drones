# HCS-SHIELD

Application web interactive de démonstration du système Celestial Integrity.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvre http://localhost:5173

## Build

```bash
npm run build
```

## Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS v4
- Framer Motion
- i18next (FR/EN)
